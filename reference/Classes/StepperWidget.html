StepperWidget

@objc public class StepperWidget: UIStepper, NumericControlViewDelegate, AlgoliaWidget

Widget that controls the Numeric value of attribute. Built on top of UIStepper.

Remark

You must assign a value to the attribute property since the refinement table cannot operate without one. A FatalError will be thrown if you don’t specify anything.
